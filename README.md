# 4-DOF-Robotic-Arm-ESP32-
A Wi-Fi controlled 4-DOF robotic arm built using servo motors and an ESP32. The system uses PWM-based servo control and an external regulated power supply for stable operation. Developed to explore embedded systems, robotic joint control, and basic kinematics through practical implementation.

HARDWARE

ESP32 Dev Board

4x SG90 Servo Motors

External 6V Battery Pack

Buck Converter (LM2596s DC- DC)

Servo Driver (PCA9685)

Jumper Wires

Acrylic Structural Frame

---------------------------------------

FEATURES

Independent control of 4 joints

PWM-based servo positioning

Wireless control using ESP32 Wi-Fi

Modular and expandable design

Stable regulated power architecture
